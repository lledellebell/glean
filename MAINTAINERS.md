## Responsibilities

### Lead Maintainer

- 프로젝트 방향 결정
- PR 리뷰 및 머지
- 릴리즈 관리
- 이슈 트리아지
- 커뮤니티 관리

### Contributors

기여자 목록은 [Contributors](https://github.com/lledellebell/glean/graphs/contributors) 페이지에서 확인할 수 있어요.

## Becoming a Maintainer

활발하게 기여하고 계신 분들 중 maintainer로 참여하고 싶으신 분은 issue를 통해 연락해주세요.

**기준:**
- 3개 이상의 머지된 PR
- 프로젝트에 대한 이해
- 지속적인 참여 의지

## Contact

- GitHub Issues: 버그 리포트, 기능 제안
- GitHub Discussions: 일반 질문, 아이디어 공유
