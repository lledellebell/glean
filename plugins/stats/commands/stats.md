# /stats - 세션 통계

코딩 세션의 통계와 생산성 분석을 보여줘요.

## 사용법
```
/stats [--period <period>] [--project <name>]
```

## 옵션
- `--period`: 기간 (today, week, month, all) - 기본: today
- `--project`: 특정 프로젝트만 필터
- `--compare`: 이전 기간과 비교

## 통계 항목

### 세션 메트릭
| 항목 | 설명 |
|------|------|
| 세션 수 | 총 세션 횟수 |
| 총 시간 | 누적 작업 시간 |
| 평균 길이 | 세션당 평균 시간 |
| 메시지 수 | 주고받은 메시지 |

### 코드 메트릭
| 항목 | 설명 |
|------|------|
| 파일 수정 | 수정한 파일 수 |
| 줄 추가/삭제 | 코드 변경량 |
| 커밋 수 | 생성한 커밋 |
| 언어별 분포 | TypeScript, JavaScript 등 |

### 도구 사용
| 항목 | 설명 |
|------|------|
| 검색 횟수 | Grep, Glob 사용 |
| 파일 읽기 | Read 호출 |
| 에이전트 실행 | Task 사용 |

### 인사이트 메트릭
| 항목 | 설명 |
|------|------|
| 발견된 패턴 | 인사이트 수 |
| 해결된 에러 | 문제 해결 |
| 학습 항목 | 새로 배운 것 |

## 출력 형식

```
## 📊 세션 통계

### 오늘 (2024-01-15)

| 메트릭 | 값 | 변화 |
|--------|-----|------|
| 세션 | 3회 | +1 |
| 시간 | 2h 30m | +45m |
| 파일 | 12개 | +5 |
| 커밋 | 4개 | +2 |

### 언어 분포
```
TypeScript  ████████████░░░░  75%
CSS         ███░░░░░░░░░░░░░  15%
Markdown    ██░░░░░░░░░░░░░░  10%
```

### 도구 사용 TOP 5
1. Read (45회)
2. Edit (32회)
3. Grep (18회)
4. Bash (12회)
5. Task (8회)

### 이번 주 트렌드
📈 생산성: +15% (저번 주 대비)
🎯 집중 세션: 2회 (1시간+ 연속)
```

## 대시보드 모드

```bash
/stats --dashboard
```

실시간 업데이트되는 대시보드 표시:
- 현재 세션 진행 상황
- 목표 대비 진척도
- 실시간 통계

## 예시

```bash
# 오늘 통계
/stats

# 이번 주 통계
/stats --period week

# 특정 프로젝트
/stats --project my-app

# 저번 주와 비교
/stats --period week --compare
```

## 데이터 저장
- `~/.glean/stats/` 에 일별 저장
- JSON 형식으로 저장
- 30일간 보관 (설정 가능)

## 연동
- `/insight` - 인사이트 통계 포함
- `/history` - 상세 세션 기록
- `/sync` - 외부 도구로 내보내기
